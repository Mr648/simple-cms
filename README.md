## About Simple Cms
This is Simple CMS created as a test project.
To use it:

  - Clone the repository
  - Make a database and edit `.env` file 
  - Run following commands: 

```
composer update
npm install
npm run dev
php artisan db:wipe
php artisan migrate --seed
```

Enjoy!
